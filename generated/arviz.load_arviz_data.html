arviz.load_arviz_data¶

arviz.load_arviz_data(dataset=None, data_home=None)[source]¶

Load a local or remote pre-made dataset.

Run with no parameters to get a list of all available models.

The directory to save to can also be set with the environement variable ARVIZ_HOME. The checksum of the dataset is checked against a hardcoded value to watch for data corruption.

Run az.clear_data_home to clear the data directory.

Parameters
datasetstr

Name of dataset to load.

data_homestr, optional

Where to save remote datasets

Returns
xarray.Dataset
